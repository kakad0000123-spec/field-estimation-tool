> **更新**：經討論後確認方向，本版為最終提案。

## 一、專案定位

這不只是一個 RC 計算工具，而是一個**可擴充的工程計算平台**。

### 第一個模組：RC 結構用量計算
- 對應 `RC_v5.xlsx`，完整複製 12 個分頁的計算邏輯
- 手機優先、離線可用、匯出 Excel

### 未來模組（已有 Excel 模板）
- **鋼構重量計算** — 對應 `鋼構重量計算模板v8.xlsx`
- **鋼構容許荷重計算** — 對應 `鋼構容許荷重計算表_v10.xlsx`
- **結構強度評估** — 待開發

### 架構設計原則
```
工程計算平台
├── 共用層（所有模組共用）
│   ├── 資料儲存引擎（IndexedDB）
│   ├── Excel 匯出引擎（SheetJS）
│   ├── 共用 UI 元件（輸入表格、計算表格、匯總頁）
│   ├── 離線快取（Service Worker）
│   └── 專案管理（新增/切換/刪除）
│
├── 模組：RC 用量計算  ← 現在做
│   ├── 專案資訊（單價、鋼筋規格）
│   ├── 9 種構件計算
│   └── RC 彙總 + 估價
│
├── 模組：鋼構重量計算  ← 未來
├── 模組：鋼構荷重計算  ← 未來
└── 模組：結構強度評估  ← 未來
```

## 二、確認的技術方案

### React + Vite（本機版，不用雲端）

| 項目 | 選擇 | 原因 |
|------|------|------|
| 框架 | React + Vite | 模組化架構好擴充、業界標準好找人維護 |
| 語言 | TypeScript | 計算公式多，型別檢查防錯 |
| 樣式 | Tailwind CSS | 手機版排版方便、不用寫很多 CSS |
| 資料儲存 | IndexedDB（Dexie.js） | 純本機、離線可用、不需伺服器 |
| Excel 匯出 | SheetJS (xlsx) | 業界標準、純前端、免費 |
| 離線 | Vite PWA Plugin | Service Worker + 可安裝到主畫面 |
| 部署 | GitHub Pages 或 Cloudflare Pages | 免費、不需伺服器 |

**白話說**：
- **不需要伺服器**、不需要帳號、不需要雲端
- 打開網址就能用，也可以「加到主畫面」像 App 一樣
- 所有資料存在手機裡，離線完全可用
- 用 React 不是為了炫技，是因為**未來要加模組時，程式碼不會亂**

### 柱（Sheet 3）
```
混凝土(m³) = B × D × H / 10⁹ × 數量
主筋(kg)   = 單位重 × (H/1000 + 搭接40D/100) × 根數 × 數量
箍筋(kg)   = 單位重 × [2×((B+D)/1000-0.08) + 2×彎鉤12D/100] × (H/間距+1) × 數量
繫筋(kg)   = 單位重 × (max(B,D)/1000-0.08 + 2×彎鉤12D/100) × 繫筋數 × (H/間距+1) × 數量
模板(m²)   = 2×(B+D) × H / 10⁶ × 數量
搭接次數   = ROUNDUP((H/1000+搭接40D/100) / 6) × 根數 × 數量
搭接長度(m)= 搭接次數 × 搭接40D/100

SD280 = 主筋(若#3~#5) + 箍筋(若#3~#5) + 繫筋(若#3~#5)
SD420 = 主筋(若#6~#11) + 箍筋(若#6~#11) + 繫筋(若#6~#11)
```

### 梁（Sheet 4）
```
混凝土(m³) = B × D × L / 10⁹ × 數量
上層筋(kg) = 單位重 × (L/1000 + 2×搭接40D/100) × 根數 × 數量    ← 兩端搭接
下層筋(kg) = 同上層筋邏輯
箍筋(kg)   = 單位重 × 周長 × [(L×0.4/密間距) + (L×0.6/疏間距) + 2] × 數量
             箍筋周長 = 2×((B+D)/1000-0.08) + 2×彎鉤12D/100
             密區 40% / 疏區 60%
模板(m²)   = (2D + B) × L / 10⁶ × 數量    ← 三面（底+兩側）
搭接次數   = [ROUNDUP((L/1000+2×搭接40D/100) / 6) - 1] × (上筋數+下筋數) × 數量
搭接長度(m)= 搭接次數 × 搭接40D/100

SD280/SD420 分類同柱，但包含上層筋、下層筋、箍筋三種
```

### 樓板（Sheet 5）
```
混凝土(m³) = 面積 × 板厚/1000 × 數量
上層筋(kg) = 單位重 / 間距 × 1000 × 2(雙向) × 1.15(搭接係數) × 面積 × 數量
下層筋(kg) = 同上層筋邏輯
點焊網(kg) = VLOOKUP(規格→單位重) × 面積 × 數量
模板(m²)   = 面積 × 數量
粉光(m²)   = 面積 × 數量
搭接次數   = ROUNDUP(√面積 / 6) × 2 × √面積/間距 × 1000 × 數量
搭接長度(m)= 搭接次數 × 搭接40D/100
```

### 牆（Sheet 6）
```
混凝土(m³) = L × H × T / 10⁹ × 數量
鋼筋(kg)   = [豎筋(L/間距 × H/1000) + 橫筋(H/間距 × L/1000)] × 單位重 × 2(雙面) × 1.15 × 數量
點焊網(kg) = VLOOKUP(規格→單位重) × L×H/10⁶ × 2面 × 數量
模板(m²)   = 2 × L × H / 10⁶ × 數量    ← 兩面

SD280/SD420 依豎筋號數分類（整組歸同一等級）
```

### 地坪（Sheet 7）
```
開挖(m³)   = 面積 × 開挖深度/1000 × 數量
PC(m³)     = 面積 × PC厚度/1000 × 數量
混凝土(m³) = 面積 × 板厚/1000 × 數量       ← 使用 fc'=210
回填(m³)   = 開挖 - PC - 混凝土
鋼筋(kg)   = 同樓板邏輯（雙向 × 1.15），鋼筋可選「---」表示無
點焊網(kg) = VLOOKUP(規格→單位重) × 面積 × 數量
模板(m²)   = 0（地坪在地面上，不需模板）
粉光(m²)   = 面積 × 數量
```

### 基礎（Sheet 8）
```
開挖(m³)   = (L+2×工作寬) × (B+2×工作寬) × (D+PC厚) / 10⁹ × 數量
PC(m³)     = (L+2×工作寬) × (B+2×工作寬) × PC厚 / 10⁹ × 數量
混凝土(m³) = L × B × D / 10⁹ × 數量
回填(m³)   = 開挖 - PC - 混凝土
鋼筋(kg)   = 單位重 × [(L方向根數×B/1000) + (B方向根數×L/1000)] × 1.2 × 數量
             L方向根數 = L/1000 / 間距 × 1000
             1.2 是基礎的搭接+彎鉤綜合係數
模板(m²)   = 2×(L+B) × D / 10⁶ × 數量
```

### 設備基礎（Sheet 9）
```
尺寸分類: 大型 / 小型（影響開挖深度和重量倍數）

開挖(m³)   = 大型: (L+2W)(B+2W)(D×0.5+PC) / 10⁹ × 數量   ← 大型半埋
             小型: (L+2W)(B+2W)(D+PC) / 10⁹ × 數量
PC(m³)     = (L+2W)(B+2W) × PC厚 / 10⁹ × 數量
混凝土(m³) = L × B × D / 10⁹ × 數量
基礎重(kg) = 混凝土體積 × 2400
需求重(kg) = 大型: 設備重×5  /  小型: 設備重×3
檢核       = 基礎重 ≥ 需求重 → "OK" / "NG"
鋼筋(kg)   = 混凝土體積 × 80 kg/m³（經驗值，全歸 SD420）
點焊網(kg) = 1.65 × (頂面積 + 4側面積) × 數量   ← 固定 phi4.0@150x150
回填(m³)   = 開挖 - PC - 混凝土
模板(m²)   = 2×(L+B) × D / 10⁶ × 數量
粉光(m²)   = L × B / 10⁶ × 數量（頂面）
```

### 樓梯（Sheet 10）
```
斜長(m)    = √[(級數×級高)² + (級數×級深)²] / 1000
混凝土(m³) = [斜長 × 梯寬/1000 × 板厚/1000 + 級數×級高×級深×梯寬/2 / 10⁹] × 數量
主筋(kg)   = 單位重 × (斜長 + 2×搭接40D/100) × (梯寬/間距) × 數量
分佈筋(kg) = 0.56 / 0.2 × 斜長 × 梯寬/1000 × 數量    ← 固定 #3@200mm
模板(m²)   = 斜長 × 梯寬/1000 × 數量（底面）
粉光(m²)   = 級數 × (級深+級高) × 梯寬 / 10⁶ × 數量（踏面+立面）
```

### 開口扣除（Sheet 11）
```
構件類型: 樓板 或 牆（影響模板扣除面數）

扣混凝土(m³) = W × H × 厚度 / 10⁹ × 數量
扣模板(m²)   = 牆: W×H×2/10⁶ × 數量（雙面）
               板: W×H/10⁶ × 數量（單面）

補強鋼筋:
  豎向(kg) = 單位重 × (H/1000 + 2×搭接40D/100) × ROUNDUP(W/原間距) × 2側 × 數量
  橫向(kg) = 單位重 × (W/1000 + 2×搭接40D/100) × ROUNDUP(H/原間距) × 2側 × 數量
  斜向(kg) = 單位重 × (√(W²+H²)/1000 + 2×搭接40D/100) × 數量   ← 僅 W≥300mm
  補強合計 = 豎向 + 橫向 + 斜向
```

### 彙總（Sheet 12）
```
9 大類匯總:
1. 混凝土: fc'=280（柱+梁+板+牆+基礎+設備基礎+樓梯-開口扣除）/ fc'=210（地坪）
2. PC 墊層: 地坪+基礎+設備基礎
3. 鋼筋: SD280 合計(噸) / SD420 合計(噸)
4. 點焊鋼絲網: 板+牆+地坪+設備基礎
5. 模板: 所有構件-開口扣除
6. 粉光: 板+地坪+設備基礎+樓梯
7. 土方: 開挖(地坪+基礎+設備基礎) / 回填(同)
8. 搭接統計: 柱+梁+板的搭接次數和長度
9. 估價: 各項數量 × 單價 → 專案總價
```
